## üîÑ What are LP Tokens?

LP (Liquidity Provider) tokens represent your share in a liquidity pool. When you deposit assets into a pool, you receive LP tokens that can be redeemed later for your share of the pool plus earned fees.

**FLIPPRX ONE supports 4 LP token pools:**

1. **XRP/FLIPPRX LP** - Main liquidity pool
2. **ECP/FLIPPRX LP** - ECP paired with FLIPPRX
3. **JNT/FLIPPRX LP** - JNT paired with FLIPPRX
4. **MFLIP/FLIPPRX LP** - MFLIP paired with FLIPPRX

## üí∞ Reserve Requirements

**Each LP trustline requires:**
- **0.2 XRP** locked as reserve
- This XRP is still yours, just locked
- You get it back if you remove the trustline

**Example:**
- 1 LP trustline = 0.2 XRP reserve
- 4 LP trustlines = 0.8 XRP reserve
- Plus 1 XRP base account reserve
- **Total minimum**: 1.8 XRP for all 4 LP trustlines

## üìä Understanding LP Tokens

### How LP Tokens Work

**When you deposit:**
```
You deposit: 100 XRP + 1000 FLIPPRX
You receive: X LP tokens (based on pool ratio)
```

**When you withdraw:**
```
You burn: X LP tokens
You receive: Your share of pool (including fees earned)
```

### LP Token Value

Your LP tokens represent:
- Your proportional share of the pool
- Accumulated trading fees
- Both assets in the pair

**Example:**
- Pool has 10,000 XRP and 100,000 FLIPPRX
- You hold 10% of LP tokens
- You own 1,000 XRP and 10,000 FLIPPRX worth of the pool

## ‚ö†Ô∏è Important Considerations

### Before Creating LP Trustlines

**Understand the risks:**

1. **Impermanent Loss**
   - Price changes can reduce value vs holding
   - More volatile pairs = higher risk
   - Fees can offset impermanent loss

2. **Authorization Required**
   - Must hold authorized NFT
   - Cannot trade without NFT
   - NFT must remain in wallet

3. **Locked Reserves**
   - 0.2 XRP per trustline
   - Cannot use reserved XRP
   - Get it back when removing trustline

4. **Transaction Costs**
   - Creating trustline: 0.2 XRP reserve
   - Deposits: Transaction fees apply
   - Withdrawals: Transaction fees apply

### Best Practices

‚úÖ **DO:**
- Start with small amounts to learn
- Understand impermanent loss
- Monitor pool statistics
- Diversify across pools
- Keep track of your positions

‚ùå **DON'T:**
- Invest more than you can afford to lose
- Ignore impermanent loss risks
- Remove trustlines with active positions
- Trade without understanding mechanics
- Forget about transaction fees

## üõ†Ô∏è Troubleshooting

### Cannot Create Trustline

**Possible reasons:**

1. **No Authorized NFT**
   - Solution: Acquire an authorized LP trading NFT

2. **Insufficient XRP**
   - Solution: Add at least 0.2 XRP for reserve

3. **Trustline Already Exists**
   - Solution: Check existing trustlines first

4. **Network Issues**
   - Solution: Wait and try again

### Trustline Not Showing

**Try these steps:**

1. Refresh the page
2. Check XRPL explorer
3. Verify transaction confirmed
4. Clear browser cache
5. Contact support if persists

### Cannot Deposit to Pool

**Check:**

1. ‚úÖ LP trustline created
2. ‚úÖ Base token trustlines created
3. ‚úÖ Sufficient token balances
4. ‚úÖ Authorized NFT in wallet
5. ‚úÖ Enough XRP for transaction fees
