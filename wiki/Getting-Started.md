## â“ Troubleshooting

### Can't See My Balance?

- Click the refresh button
- Check your internet connection
- Verify you're on the correct network
- Clear browser cache

### Transaction Failed?

**Common reasons:**
- Insufficient balance (including reserves)
- Invalid destination address
- Missing trustline
- Network congestion

### Forgot Password?

**With Seed Phrase:**
1. Create new wallet
2. Import using seed phrase
3. Set new password

**Without Seed Phrase:**
- Unfortunately, funds cannot be recovered
- This is the nature of self-custodial wallets

### SnapTap Not Working?

- Ensure browser supports WebAuth
- Check NFC is enabled (if using NFC)
- Try biometric authentication instead
- Update your browser
- Use password as fallback
